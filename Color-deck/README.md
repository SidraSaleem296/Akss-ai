# AKSS AI Color Deck

Static design deck for the proposed AKSS AI rebrand around `#152D35`, silver-white, and black.

## Pages

- `index.html` - overview and comparison cards
- `01-velvet-command.html` - strongest dark velvet route
- `02-silver-glass.html` - lighter silver-white trust route
- `03-onyx-velvet.html` - premium black-glass route
- `04-velvet-mint.html` - warmer velvet-mint route

## Core Colors

- Velvet green: `#152D35`
- Deep velvet: `#0F2026`
- Black: `#050607`
- Silver white: `#F6F8F8`
- Silver: `#D9DEE2`
- Mist: `#B8CBC4`

Open `index.html` in a browser to compare the concepts.
